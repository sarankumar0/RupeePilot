# RupeePilot

A personal finance assistant for young Indian professionals and freelancers.

## What it does
- Track expenses via Telegram bot
- AI categorises your spending automatically
- Web dashboard with charts and insights
- Goal planning and investment guidance (coming later)

## Tech Stack
- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS
- **Backend:** NestJS + TypeScript + Mongoose
- **Database:** MongoDB
- **AI:** Groq API (llama3-70b)
- **Bot:** Telegram Bot API
